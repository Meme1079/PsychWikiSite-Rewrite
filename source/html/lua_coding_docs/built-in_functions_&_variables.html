      string.byte()

      Converts any alphabetical characters into ASCII characters.

      Parameters Description Data Type Defualt Value Speciality
      str The characters from the string to be converted. String None None
      startPos The specified starting position from the string to convert. Int 1 Optional
      endPos The specified ending position from the string to convert. Int 1 Optional 
      function onCreate()
     debugPrint(('a'):byte())
end

      > main.lua L-2: 97

      Syntax 

      string.byte(str, startPos, endPos)

      (str):byte(startPos, endPos)

      Return Values

      Data Type Description
      Int The character representing to its corresponding ASCII character. If the startPos and endPos arguments are used, it will return mutiple values. If you want to get the other values convert it into a table.

      string.char()

      Converts any ASCII characters into alphabetical characters.

      Note: This function doesn't support the colon operator shortcut!

      byte The ASCII character from the string to be converted. Int None Infinite

      string.rep()

      Repeats the string's content and concatenated them together—determined by how many times it will repeat itself.

      Parameters Description Data Type Defualt Value Speciality
      str The string's content to be repeated. String None None
      repeat How many times it will be repeated. Int None None 
      function onCreate()
     debugPrint(('repeated-'):rep(3))
end
